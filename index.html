Girls Coode

Bem-vindo à "Girls Code". Somos uma plataforma dedicada a capacitar mulheres através da educação em tecnologia. Em um mundo onde a igualdade de gênero no campo da tecnologia ainda é um desafio, estamos comprometidos em criar oportunidades acessíveis e inclusivas para mulheres de todas as origens.

Sobre Nós

Na Girls Code, nosso compromisso é claro: capacitamos mulheres através da educação em tecnologia, promovendo a igualdade de gênero em um campo que historicamente tem sido dominado por homens. Desde a nossa fundação, buscamos quebrar barreiras e oferecer oportunidades inclusivas para mulheres de todas as origens.

Nossos Objetivos

Nossos objetivos são simples, mas poderosos. Queremos proporcionar às mulheres as ferramentas e conhecimentos necessários para prosperar na indústria de tecnologia, capacitando-as com habilidades em programação, decodificação e outras áreas afins. Além disso, buscamos criar uma comunidade de apoio, onde mulheres possam se conectar, colaborar e se inspirar mutuamente.

O Que Fazemos

Na Girls Code, oferecemos uma ampla gama de cursos e recursos educacionais, adaptados para atender às necessidades de mulheres em diferentes estágios de suas jornadas tecnológicas. Desde cursos introdutórios para iniciantes até programas avançados para profissionais experientes, temos algo para todos. Além disso, organizamos eventos, palestras e hackathons, proporcionando oportunidades únicas para aprendizado, networking e crescimento profissional.

Curso

Curso Técnico de Desenvolvimento Web - Exclusivo para Mulheres

Visão Geral do Curso

O Curso Técnico de Desenvolvimento Web, exclusivo para mulheres, tem como objetivo capacitar alunas para atuarem como desenvolvedoras web completas, com habilidades robustas em Front End, Back End e Banco de Dados. Este curso foi pensado para fornecer um ambiente de aprendizado inclusivo e empoderador, incentivando a presença feminina na tecnologia.

Estrutura do Curso


Duração: 1 ano (com possibilidade de estágios e projetos práticos)
Carga Horária: 1200 horas
Modalidade: Presencial

Conteúdo Programático

Front End Development

No módulo de Front End, as alunas aprenderão a criar interfaces de usuário intuitivas e responsivas. O foco será nas seguintes áreas:
- HTML5: Estruturação básica de páginas web.
- CSS3: Estilização de elementos HTML para criar designs atrativos e responsivos.
- Flexbox e Grid Layout
- Animações e Transições
- JavaScript: Interatividade e dinâmica nas páginas web.
- Manipulação do DOM
- Eventos e Listeners
- ES6+ (incluindo Arrow Functions, Promises e Async/Await)
- Frameworks e Bibliotecas: Ferramentas modernas para desenvolvimento eficiente.
- React.js: Criação de interfaces de usuário componíveis e reativas.
- Bootstrap: Framework de CSS para desenvolvimento ágil e responsivo.

Back End Development

No módulo de Back End, as alunas aprenderão a construir a lógica e funcionalidade por trás de um site ou aplicativo web. O foco será nas seguintes áreas:
- Node.js: Plataforma para execução de código JavaScript no servidor.
- Express.js: Framework minimalista para criação de servidores web.
- APIs RESTful: Criação e consumo de APIs para comunicação entre front end e back end.
- Autenticação e Autorização: Implementação de sistemas seguros de login e controle de acesso.
- JWT (JSON Web Tokens)
- OAuth
- Gestão de Dependências e Ferramentas de Build:
- npm (Node Package Manager)
- Webpack e Babel

Banco de Dados

No módulo de Banco de Dados, as alunas aprenderão a projetar, implementar e gerenciar bancos de dados eficientes e seguros. O foco será nas seguintes áreas:
- Banco de Dados Relacionais:
- SQL: Linguagem de consulta estruturada para bancos de dados relacionais.
- MySQL: Sistema de gerenciamento de banco de dados relacional.
- Normalização de Dados: Técnicas para organizar dados de forma eficiente.
- Banco de Dados Não-Relacionais:
- MongoDB: Banco de dados NoSQL orientado a documentos.
- ORM (Object-Relational Mapping):
- Mongoose para MongoDB
- Sequelize para SQL

Projetos Práticos e Estágios

Ao longo do curso, as alunas terão a oportunidade de aplicar os conhecimentos adquiridos em projetos práticos e estágios supervisionados. Estes projetos incluem:
- Desenvolvimento de um site completo com funcionalidades front end e back end.
- Criação de uma API RESTful para um aplicativo de gerenciamento de tarefas.
- Implementação de um sistema de banco de dados para uma aplicação de comércio eletrônico.
